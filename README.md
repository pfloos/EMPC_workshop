# EMPC_workshop

Workshop on "Embedding in Math, Physics and Chemistry".

This repository contains the workshop materials (HTML pages) and supporting build helpers.

Homepage

- https://pfloos.github.io/EMPC_workshop

What you'll find

- Slides, notebooks, and examples exported as HTML
- Build helpers and automation (Makefile)

Quick start

- Open the main HTML file in a browser (for example, open `index.html` in your browser).
- Or serve the site locally:

```sh
python -m http.server 8000
# then visit http://localhost:8000
```
